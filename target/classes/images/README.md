# NeuraSys JavaFX UI - Images Folder

This folder contains image assets for the JavaFX user interface.

## Required Images

- **icon.png** (256x256) - Application window icon
    - Format: PNG with transparency
    - Theme: Technology/System Monitoring/Backup related

- **logo.png** (512x256) - Application logo for splash screen
    - Format: PNG
    - Text: "NeuraSys" with tagline
    - Colors: #2c3e50 (dark), #27ae60 (green)

## Optional Images

- **background.jpg** - Dashboard background
- **empty-state.png** - Empty state illustrations
- **tutorial-*.png** - Tutorial/help images

## Notes

- Images are optional for development; app works without them
- Place PNG files directly in this folder
- Update FXML references in `main.fxml` to match image filenames if added
